Help on Keywordsearch

The Open Computer Forensics Architecture uses the Lucene indexer for Information Retrieval purposes. Lucene gives the opportunity to search one or more selected indexes for keywords and give all documents containing the given keywords. Please feel free to consult the Lucene help pages. This page is only a short page containing some hints.

Query Parser Syntax

The Query Parser Syntax describes the possibilities to combine and extend a search string by using fields or wildcards.

Fields

  • Use 'AND' to search for all keywords.
    Eg.: The search for "gun AND drugs" returns only the documents containing both words.
  • Use 'OR' to search for one of the given keywords.
    Eg.: The search for "gun OR riffle" returns all documents containing one of the given terms or both terms.

Wildcard Search

  • Use the '?' character to match exactly one unknown character
  • Use the '*' character to match more than one unknown character
